Dans ce dossier, vous trouverez le code PHP d'une plate-forme web pour accéder à une base de données MySQL online.

Il y a 9 pages php, ainsi qu'une page de style et le fichier avec la base de données.

Pour utiliser ces fichiers, importez le fichier .sql dans votre base de données MySQL et connectez-vous sur la page PHP avant de naviguer sur les différentes pages.
